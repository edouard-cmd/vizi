Documentation d'usage

Règles courtes, applicables. Pour designers + dev.

Règles d'usage

  • Taille minimale : 14px. Sous 14px, retirer les éléments d'opacité < 0.6 (ils disparaissent).
  • Espacement : 8px minimum autour, 12px à 24px+, 16px à 32px+.
  • Stroke : 1.5px à 24px, 2px à 32-48px, 2.4px sur marqueurs carte.
  • Fond Talisker : couleur #4DD4A8 (teal) par défaut.
  • Fond satellite : dans pastille blanche (taille 36-44px) ou ring deep+teal.
  • Pas de gradient, pas de filtre, pas de drop-shadow sur l'icône elle-même.

Convention de couleurs

  • #4DD4A8 teal · couleur par défaut, UI standard
  • #2DA888 teal-mid · états positifs secondaires
  • #D8C84A jaune · visi moyenne, warning faible
  • #E89B3C orange · warning, visi faible
  • #C94A3D rouge · danger, visi nulle
  • Blanc #FFFFFF · sur marqueurs blancs uniquement

Intégrer une nouvelle icône

<svg width="24" height="24" 
  viewBox="0 0 24 24"
  fill="none"
  stroke="currentColor"
  stroke-width="1.5"
  stroke-linecap="round"
  stroke-linejoin="round">
  <path d="..."/>
</svg>
  • Utiliser currentColor sur stroke (héritage couleur parent)
  • Courbes Q/C uniquement (pas de segments droits sauf nécessité)
  • Hiérarchie : stroke-width="2" primaire, 1.5 secondaire avec opacity
  • Aire utile : 22×22 sur viewBox 24, 1px de marge minimum
